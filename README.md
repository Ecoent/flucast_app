# flucast_app

A OpenSource Podcast app for your Show

## Getting Started

This project is a starting point for a Flutter application.

## Features

FluCast has the most features that you need to play your show:

- Podcast Cover
- Podcast Description
- Episodes Listing
- Player Status
- Play, Pause and Stop
- 30 Seconds Seek
- Episode Description
